Welcome to Reynolds's documentation!
====================================

`Reynolds`_ is a full featured, scriptable python API of components for the
preprocessing and solver environments of `OpenFoam`_. These components can be
easily combined to build a GUI using any 3D graphics package such as `Blender`_.

.. _Reynolds: https://github.com/dmsurti/reynolds
.. _OpenFoam: http://www.openfoam.com
.. _Blender: http://www.openfoam.com
